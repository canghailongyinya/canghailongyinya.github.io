MIPS多周期CPU设计

Created2025-01-16|Updated2025-01-16|计算机组成原理
|Word Count:455|Reading Time:2mins|Post Views:

前言

  • 书接上回：MIPS单周期CPU
  • 在单周期的基础完成多周期CPU的设计只需要改进很少的地方

一、控制信号

  • 由于多周期CPU需要多个周期才能完成一个指令，每条指令的周期数不同，因此需要设计额外的控制信号来控制标志不同指令需要执行的阶段。
  • 每条指令执行的阶段如下，1代表有，0代表无。
指令\阶段 IF ID EX MEM WB
add 1 1 1 0 1
sub 1 1 1 0 1
and 1 1 1 0 1
or 1 1 1 0 1
slt 1 1 1 0 1
sll 1 1 1 0 1
addi 1 1 1 0 1
andi 1 1 1 0 1
ori 1 1 1 0 1
lw 1 1 1 1 1
sw 1 1 1 1 0
beq 1 1 1 0 0
bne 1 1 1 0 0
j 1 1 0 0 0
halt 1 1 0 0 0
  • 可见多周期CPU的所有指令都有IF、ID阶段，所以我们只需要设计额外的三位控制信号来标志某条指令是否具有EX、MEM、WB阶段即可。
